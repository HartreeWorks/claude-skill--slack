# Slack

A Claude Code skill for interacting with Slack via a self-contained Python client, supporting multiple workspaces, messaging, activity digests, and message history.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

```bash
npx skills add HartreeWorks/skill--slack
```

Or install all HartreeWorks skills: `npx skills add HartreeWorks/skills`
